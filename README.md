Tech Stack:

1. HuggingFace --> LLM and Embedding Model.
2. FAISS --> Local Vector Store for storing your embedings.
3. Langchain --> Generative AI Framework to interact with LLM.
4. PyPDF --> Library for reading contents of a PDF file.
5. Flask --> To build backend API's for handling user requests.
6. HTML/CSS --> To make UI or frontend of the app.
7. Docker --> For Containerization of the app during deployment.
8. Aqua Trivy --> For scanning Docker images for security issues and vulnerabilities.
9. Jenkins --> For making CI/CD Pipelines.
10. AWS Runner --> To deploy and run your app on cloud with ease and reliability. It's a service offered by AWS.
11. GitHub --> It will work as a SCM for your project.
